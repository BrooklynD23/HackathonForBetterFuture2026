**Step 1: Add to `frontend/src/lib/api.ts`:**

Add these types near the other interfaces (before the `requestJson` function):

```typescript
export interface CrawlerEvent {
  url: string;
  title: string;
  status: "crawling" | "found" | "error" | "done";
  timestamp: string;
}

export interface CrawlerResultsResponse {
  events: Array<Record<string, unknown>>;
  count: number;
  source: string;
}
```

Add these functions near the other fetch functions:

```typescript
export async function startCrawl(): Promise<{ status: string }> {
  return requestJson<{ status: string }>("/api/crawler/start", {
    method: "POST",
  });
}

export async function fetchCrawlerResults(): Promise<CrawlerResultsResponse> {
  return requestJson<CrawlerResultsResponse>("/api/crawler/results");
}
```

**Step 2: Create `frontend/src/components/CrawlerFeed.tsx`:**

Build a self-contained component with these characteristics:

1. **Props interface:**
```typescript
interface CrawlerFeedProps {
  className?: string;
}
```

2. **Internal state:**
   - `events: CrawlerEvent[]` — accumulated feed events (newest first, max 50)
   - `isActive: boolean` — whether a crawl is in progress
   - `isDone: boolean` — whether crawl finished
   - `foundCount: number` — count of "found" status events

3. **useCrawlerFeed hook (inline or extracted):**
   - When `isActive` becomes true, create `new EventSource("/api/crawler/feed")`
   - Store ref in `useRef<EventSource | null>(null)`
   - `es.onmessage = (e) => { const data = JSON.parse(e.data); ... }`
   - If `data.status === "done"`: set isDone=true, close EventSource, return
   - Otherwise: prepend to events array (newest first), cap at 50 entries with `.slice(0, 50)`
   - `es.onerror = () => { es.close(); setIsActive(false); }`
   - Cleanup: `return () => { esRef.current?.close(); }`
   - Dependency array: `[isActive]`

4. **handleStartCrawl handler:**
   - Call `startCrawl()` from api.ts
   - Set `isActive(true)`, `isDone(false)`, `events([])`

5. **Render structure (using Tailwind + shadcn Card):**
```tsx
<Card className={className}>
  <CardHeader className="pb-3">
    <div className="flex items-center justify-between">
      <CardTitle className="text-sm font-medium flex items-center gap-2">
        <Globe className="h-4 w-4" />
        Web Crawler Feed
      </CardTitle>
      <Button
        size="sm"
        variant={isActive ? "outline" : "default"}
        onClick={handleStartCrawl}
        disabled={isActive}
      >
        {isActive ? "Crawling..." : "Start Crawl"}
      </Button>
    </div>
  </CardHeader>
  <CardContent>
    {isDone && (
      <div className="mb-3 rounded-md bg-green-50 px-3 py-2 text-sm text-green-700">
        Found {foundCount} directed school pages — results saved
      </div>
    )}
    <div className="max-h-[320px] overflow-y-auto space-y-1.5">
      {events.length === 0 && !isActive && (
        <p className="text-sm text-muted-foreground py-4 text-center">
          Click "Start Crawl" to discover IA West directed school pages
        </p>
      )}
      {events.map((event, i) => (
        <div
          key={`${event.timestamp}-${i}`}
          className="flex items-start gap-2 rounded-md px-2 py-1.5 text-sm animate-in slide-in-from-bottom-2 duration-300"
        >
          <span className="mt-0.5 shrink-0">
            {event.status === "crawling" && <Loader2 className="h-3.5 w-3.5 animate-spin text-blue-500" />}
            {event.status === "found" && <CheckCircle2 className="h-3.5 w-3.5 text-green-500" />}
            {event.status === "error" && <XCircle className="h-3.5 w-3.5 text-red-400" />}
          </span>
          <div className="min-w-0 flex-1">
            <span className="font-medium truncate block">
              {event.title || event.url}
            </span>
            {event.url && event.title && (
              <span className="text-xs text-muted-foreground truncate block">
                {event.url}
              </span>
            )}
          </div>
        </div>
      ))}
    </div>
  </CardContent>
</Card>
```
