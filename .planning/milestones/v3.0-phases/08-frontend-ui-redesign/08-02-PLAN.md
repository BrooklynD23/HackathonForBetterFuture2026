Create `src/ui/data_helpers.py` with these functions:

1. **`_data_dir() -> Path`** — returns `Path(__file__).resolve().parent.parent.parent / "data"` (the `data/` directory relative to the project root). Use `pathlib.Path`.

2. **`load_specialists() -> list[dict[str, str]]`** — reads `data_speaker_profiles.csv` using `csv.DictReader`. Returns list of dicts with keys: `name`, `board_role`, `metro_region`, `company`, `title`, `expertise_tags`, `initials`. The `initials` field is computed from the `Name` column using `get_initials()`. Cache with `@functools.lru_cache(maxsize=1)` (wrap in a tuple-returning inner function).

3. **`load_poc_contacts() -> list[dict]`** — reads `data/poc_contacts.json` using `json.load`. Returns the parsed list as-is. Cache with lru_cache.

4. **`load_pipeline_data() -> list[dict[str, str]]`** — reads `data/pipeline_sample_data.csv` using `csv.DictReader`. Returns list of dicts with keys matching CSV columns. Cache with lru_cache.

5. **`load_event_calendar() -> list[dict[str, str]]`** — reads `data/data_event_calendar.csv` using `csv.DictReader`. Returns list of dicts. Cache with lru_cache.

6. **`get_initials(name: str) -> str`** — splits name on spaces, takes first letter of each part, uppercases, joins. E.g., `"Travis Miller"` -> `"TM"`, `"Dr. Yufan Lin"` -> `"DYL"`.

7. **`get_top_specialists_for_event(event_name: str, limit: int = 3) -> list[dict]`** — filters pipeline_sample_data for rows matching `event_name`, sorts by `match_score` descending, takes top `limit`. For each, joins with specialist data from `load_specialists()` to enrich with `company`, `title`, `expertise_tags`, `initials`. Returns list of dicts with keys: `name`, `match_score`, `rank`, `stage`, `company`, `title`, `expertise_tags`, `initials`.

8. **`get_recent_poc_activity(limit: int = 5) -> list[dict]`** — loads POC contacts, flattens all comm_history entries across contacts, sorts by date descending, returns top `limit` entries. Each entry has: `poc_name`, `org`, `date`, `type`, `summary`.

All functions use type annotations. Use `from __future__ import annotations`. Module docstring explains purpose.

Error handling: If a data file is missing, return an empty list and log a warning via `logging.getLogger(__name__).warning(...)`.
